MidTerm Mobile Device Bridge

1. Extract this ZIP to a permanent local folder.
2. Open chrome://extensions in Chrome.
3. Enable Developer mode and choose Load unpacked.
4. Select the extracted folder.
5. Open MidTerm, then click the extension icon in that tab (or press Ctrl+Shift+Y).

The extension runs Chrome device emulation on this computer. MidTerm itself may run on a remote host.
Chrome shows a debugger notice while a device window is attached; that is expected.
